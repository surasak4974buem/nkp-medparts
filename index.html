เชื่อมต่อฐานข้อมูล Google Sheets

ระบบดึงและบันทึกข้อมูลผ่าน Google Apps Script Web App ที่เชื่อมต่อกับ Google Sheets ของคุณ กรุณาตั้งค่า URL ด้านล่าง เพื่อเริ่มใช้งานระบบได้ทันที

URL นี้จะได้มาหลังจากอัปโหลดไฟล์ Code.gs และ Deploy เป็น Web App ใน Google Apps Script แล้ว

วิธีเริ่มติดตั้งระบบ

🔧 แนะนำสำหรับผู้ดูแลระบบ: ทำตามขั้นตอนนี้เพื่อเชื่อมต่อ Google Sheets เป็นฐานข้อมูลหลัก
  1. สร้างไฟล์ Google Sheet ใหม่ใน Google Drive
  2. ไปที่เมนู Extensions > Apps Script
  3. คัดลอกซอร์สโค้ดจากไฟล์ google-apps-script/Code.gs ไปวางใน Script Editor
  4. กด Deploy > New Deployment
  5. เลือกประเภท Web App:
    • Execute as: Me (อีเมลของคุณ)
    • Who has access: Anyone
  6. กด Deploy และอนุมัติสิทธิ์ (Authorize Access)
  7. คัดลอก Web App URL มาวางในช่องด้านซ้าย แล้วกดบันทึก
