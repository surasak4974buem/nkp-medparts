ตั้งค่าการเชื่อมต่อฐานข้อมูล Google Sheets

ระบบนี้เป็นเว็บบอร์ดกึ่งออฟไลน์ ทำการดึงและเขียนฐานข้อมูลของคุณใน Google Sheets ผ่าน Google Apps Script Web App URL

คุณจะได้ URL นี้หลังจากอัปโหลดไฟล์ Code.gs ขึ้น Apps Script ใน Google Sheet และกด Deploy เป็น Web App

วิธีเริ่มติดตั้งระบบ Google Sheets Database

  1. สร้างไฟล์ Google Sheet ขึ้นมาใหม่ใน Google Drive
  2. ไปที่เมนู Extensions > Apps Script
  3. คัดลอกซอร์สโค้ดจากไฟล์ google-apps-script/Code.gs ในเครื่องของคุณ ไปวางลงใน Script editor ของ Google Apps Script
  4. กดปุ่ม Deploy > New Deployment
  5. เลือกประเภท Web App ตั้งค่าดังนี้:
    • Execute as: Me (อีเมลของคุณ)
    • Who has access: Anyone (ทุกคน รวมถึงบุคคลภายนอก)
  6. กด Deploy และทำการอนุมัติสิทธิ์ (Authorize Access) ให้กับสคริปต์
  7. คัดลอก Web App URL ที่ปรากฏหลัง Deploy สำเร็จ นำมาวางในช่องด้านบน แล้วกดบันทึก
